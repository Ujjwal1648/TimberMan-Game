# TimberMan Game

A simple Timberman-style game developed using C++ and SFML.

## Features
- Player movement
- Tree chopping gameplay
- Score tracking
- Sound effects

## How to Run

Compile:
g++ test.cpp -o game -lsfml-graphics -lsfml-window -lsfml-system -lsfml-audio

Run:
./game
